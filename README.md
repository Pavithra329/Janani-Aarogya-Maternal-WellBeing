## 🩺 Janani Aarogya: A Comprehensive Maternal Healthcare Platform  

**Janani Aarogya** is a digital healthcare platform designed to support women during pregnancy and post-pregnancy.  
It integrates community support, mental well-being tools, and AI-powered personalization to empower mothers through every stage of their maternal journey.  

The platform provides features like an **AI-powered chatbot** for instant maternal healthcare assistance, **medicine purchase options**, **daily motivational content**, **expert health advice**, and **gamification with rewards** to encourage wellness.  
It also offers **interactive community forums** where mothers can share experiences and support each other, creating a holistic ecosystem for maternal care.  

---

## ✨ Features

- **AI-powered Chatbot** – Provides instant assistance and answers to maternal healthcare queries.  
- **Medicine Purchase** – Integrated feature to order and manage medicines conveniently.  
- **Daily Motivation** – Delivers daily motivational quotes and wellness reminders for mothers.  
- **Expert Health Advice** – Access to professional guidance from healthcare experts.  
- **Gamification & Rewards** – Interactive games and a points system to encourage maternal wellness.  
- **Community Forums** – Safe space for mothers to share experiences, stories, and support each other.  
- **Pregnancy & Post-Pregnancy Support** – Resources and tools tailored for mothers during pregnancy and after childbirth.  

---

## 🛠️ Tech Stack

- **React.js** – Frontend framework for building a responsive, interactive, and accessible user interface.  
- **Node.js** – Runtime environment for executing server-side logic.  
- **Express.js** – Backend framework for building RESTful APIs and managing server-side operations.  
- **MongoDB** – NoSQL database for securely storing and managing user data and healthcare records.  
- **Chatbot Integration** – Provides instant maternal healthcare assistance through conversational AI.  
- **Web Accessibility Support** – Ensures the platform is inclusive and usable by all users.

## 📦 Requirements  

Make sure you have the following installed before running the project:  

- Node.js (v16+)  
- npm or yarn package manager  
- MongoDB (local or cloud instance)  

To install all dependencies, run:  
```bash
npm install
```

To start the development server:
```bash
npm start
```
